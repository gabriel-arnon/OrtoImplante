# Teste Manual do Formulario

Executar somente com dados ficticios.

1. Abrir a pagina inicial.
2. Preencher nome ficticio, telefone ficticio, cidade ficticia, categoria e descricao curta.
3. Confirmar que nenhum campo solicita CPF, RG, senha, token, documento, extrato ou upload.
4. Confirmar que falhas preservam os dados digitados.
5. Confirmar que sucesso aparece somente quando o modo de entrega aceitou a requisicao.
