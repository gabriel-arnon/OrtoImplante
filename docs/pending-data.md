# Dados Pendentes

Itens que exigem decisao humana antes de desenvolvimento publico:

- identidade final;
- dominio final;
- canais de contato;
- enderecos;
- textos institucionais;
- politica de privacidade final;
- imagens e icones aprovados;
- provedor de e-mail;
- rate limiting externo;
- aprovacao de indexacao.
